FlowMark is a Go-based benchmarking tool built to simulate blockchain transaction loads under varied network conditions and generate performance reports.

It came out of a hackathon setting, but the project fits the broader pattern of my work: build a harness, generate pressure, measure behavior, and use the results to reason about system limits.

**What it shows**

- Go backend tooling for load generation and performance reports.
- High-concurrency transaction simulation.
- Benchmarking instincts that transfer beyond blockchains into model-serving and infrastructure systems.
- Second place in the Flow Blockchain Hackathon.
